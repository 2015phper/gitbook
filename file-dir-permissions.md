#### Linux 的文件类型

1. 普通文件，符号为 `-`
2. 目录，符号为 `d`
3. 字符设备，符号为 `c`
4. 块设备，符号为 `b`
5. 套接字文件，符号为 `s`
6. 软链接文件，类似 ·Windows· 系统的快捷方式，删除了并不会影响源文件，符号为 `l`
7. 管道文件，符号为 `p`

补充：常见的文件类型为 `-` `d` `l`

#### Linux 的文件权限

- `r` 代表 `read` 读权限，数字代表为 `4`

- `w` 代表 `wirte` 写权限，数字代表为 `2`

- `x` 代表 `execute` 执行权限，数字代表为 `1`
